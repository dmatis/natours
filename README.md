## Key Features

### Modular

-   Implements the [7-1 Pattern](https://sass-guidelin.es/#the-7-1-pattern) in Sass

### Responsive

-   Fonts are all rem based
-   Image widths are %-based for multiple devices

### Dynamic

-   Page load animations
-   Custom button animations
